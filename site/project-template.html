Obskewer
A cipher tool




What is Obskewer?

Obskewer is a light-weight Java application for encoding and decoding text using common ciphers.


Supported Ciphers:

  • ClassicShift
  • The Caesar Cipher
  • Morse code
  • Letter-Numbers
  • "Skip" (Scytale)(coming soon)
