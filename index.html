about

Welcome to ruthlessphysics.com.
Hi, I am Austin Jackson (MacPhage) and I'm a programmer in high school. Here I'll have some of my projects available for public use and criticism.

Being an Apple fanatic, I try to make my software run on as many desktop platforms as possble, including OS X. Some of my software may require emulation, such as using WineBottler. My software is provided as-is and is not harmful to your computer, but wether you trust me or not is ultimately up to you.

projects

Obskewer

Obskewer is a light-weight Java application for encoding and decoding text using common ciphers.


pCalculator

pCalculator is a light-weight Java application for convenient calculation. It is designed to be as fully functional and do-all-the-work-for-you as possible.


pDraw

pDraw is a WinBGIm (graphics.h) C++ painting application used to create digitally-primitive drawings using your mouse and keyboard.


pTicTacToe

pTicTacToe is a WinBGIm (graphics.h) C++ program that allows two users to play a game of 3-by-3 TicTacToe against each other.
